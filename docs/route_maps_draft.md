# Черновик маршрутных карт изделий

Это каркас для проверки. Бот пока не использует эти маршруты в меню сотрудников.

## Общий старт для всех изделий

1. Раскройщик: Нанесение контуров лекал на ткань -> Контуры нанесены
2. Раскройщик: Формирование настила -> Настил сформирован
3. Раскройщик: Раскрой -> Раскроено
4. Раскройщик: Формирование готового кроя -> Готовый крой

## Общий финиш для всех изделий

1. Упаковщик: Подготовка к упаковке -> Подготовлено к упаковке
2. Упаковщик: Упаковка -> Упаковано
3. Упаковщик: Размещение на склад -> На складе

## Брюки со стрелками детские

1. Упаковщик: Заутюживание стрелок -> Стрелки заутюжены
2. Швея: Отстрочка стрелок -> Стрелки отстрочены
3. Упаковщик: ВТО пошитых стрелок -> Стрелки после ВТО
4. Швея: Сборка брюк на оверлоке -> После оверлока
5. Швея: Притачивание резинки к поясу -> Резинка притачана
6. Швея: Отстрачивание пояса -> Пояс отстрочен
7. Швея: Формирование низа брюк -> Низ сформирован

## Брюки со стрелками подростковые

2. Швея: Подготовка кармана на оверлоке -> Карман подготовлен
3. Швея: Отстрочка стрелок -> Стрелки отстрочены
4. Швея: Формирование кармана -> Карман сформирован
5. Упаковщик: ВТО пошитых стрелок -> Стрелки после ВТО
6. Швея: Сборка брюк на оверлоке -> После оверлока
7. Швея: Стачивание пояса в кольцо -> Пояс стачан в кольцо
8. Швея: Пришивание пояса к брюкам -> Пояс пришит
9. Швея: Формирование низа брюк и закрепки на поясе -> Низ сформирован, закрепки выполнены

## Что нужно проверить

1. У всех ли изделий общий старт через раскрой верный.
2. Всем ли изделиям нужен общий финиш: подготовка к упаковке, упаковка, размещение на склад.
3. Верно ли, что ВТО стрелок выполняет упаковщик, а не швея.
4. Верно ли, что дублирование относится к упаковщику и стоит первым после раскроя для кардиганов, бомбера и жакета.
5. Нужны ли параллельные подготовительные операции: нарезание резинки, сшивание резинки, нарезание дублерина.
